How this demo meets the job expectations

I scraped the live celltheory.com Framer site and product page, identified the existing UTM script, and built a solution that fixes its flaws. Here is how each expectation is satisfied:

  • Analyze the existing setup — Verified the Framer frontend at celltheory.com, the Shopify/Seal product page at /my-store/, and the existing UTM handler that only decorates shop.celltheory.com links and uses non-cross-domain sessionStorage.
  • Implement reliable UTM preservation — Uses three redundant mechanisms: URL decoration on outbound links, a root-domain .celltheory.com cookie, and localStorage. New URL params always win.
  • Works with existing GTM — Pushes a utm_parameters_captured event to window.dataLayer with all five UTM values, so GTM tags can fire immediately on both domains.
  • Handles Seal Subscriptions — The subscription checkout step reconstructs UTM params from the decorated URL and the shared root-domain cookie, so subscription conversions keep their attribution.
  • Documentation & guidance — The README and this page explain the problem, the implementation, and how to apply it to the real site.
